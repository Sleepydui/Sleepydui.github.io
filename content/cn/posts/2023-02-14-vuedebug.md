- 问题
  - 从GitHub中下载的vue工程用npm run serve无法打开
  - npm run serve后报错vue-cli-service:command not found

- 方法
  - 排查全局环境问题——我自己的工程文件是可以用npm run serve打开的
  - 排查文件夹node_modules问题
  - 最终解决方法
    1. rm -rf node_modules  #删除GitHub中下载下来的node_modules文件夹
    2. npm cache clean --force  #清除缓冲
    3. npm install  #重新安装
